# 上位机软件设计文档 v2.0

## 1. 概述
本项目是一款用于单光子激光雷达数据接收、实时显示、记录及离线分析的 Windows 上位机软件。
软件采用 Python (PyQt5 + PyQtGraph) 开发，专注于高性能图像渲染和数据流处理，界面采用现代 Linux 深色风格 (Fusion Dark Theme)。

## 2. 需求规格

### 2.1 输入接口
- **协议**：UDP
- **网络配置**：可配置 IP 和端口 (默认 127.0.0.1:5005)。
- **数据内容**：
    - **Type 0 (强度+距离)**：双通道图像，128x128 像素。
    - **Type 1 (ToF)**：光子飞行时间数据，128x128 像素，16-bit 深度 (0-16000)。
- **帧率**：约 50Hz。

### 2.2 数据格式 (详细见 udp.md)
- **UDP 包结构**：协议头 (AA55) + 控制字 + ID + 类型 + 序号 + 数据负载 (4096 字节) + 校验。
- **帧组装**：
    - **深度/强度帧**：65536 字节/帧 (16个 UDP 包)。
    - **ToF 帧**：32768 字节/帧 (8个 UDP 包)。

### 2.3 功能模块

#### A. 数据接收与解析 (Receiver)
- **多线程接收**：独立线程处理 UDP Socket，避免阻塞 UI。
- **分片重组**：根据 Frame ID 和 Seq 序号重组 UDP 分片。
- **类型识别**：自动识别 Type 0 和 Type 1 数据包并分发到对应处理逻辑。

#### B. 实时显示 (Visualization)
- **界面布局**：
    - **左侧显示区 (70%)**：
        - **Tab 1 (强度/距离)**：2x2 布局 (强度图+直方图，距离图+直方图)。
            - 强度图：灰度显示 (Grayscale)。
            - 距离图：伪彩色显示 (Color)。
        - **Tab 2 (ToF)**：ToF 图像 + 直方图 (伪彩色)。
    - **右侧配置区 (30%)**：多 Tab 结构 (基础配置、算法设置、离线分析)。
- **交互功能**：
    - **鼠标交互**：点击图像任意位置，显示像素坐标及数值。
    - **显示控制**：支持手动调整数值映射范围 (Min/Max SpinBox)。
    - **FPS 统计**：左上角实时显示刷新率。

#### C. 数据记录 (Recorder)
- **存储策略**：
    - **纯净存储**：仅保存有效数据负载 (Payload)，去除网络协议头。
    - **类型锁定**：一次录制仅锁定一种数据类型 (首帧决定)，自动丢弃其他类型数据。
- **文件命名**：
    - **自动命名**：`depth_YYYYMMDD_HHMMSS.bin` 或 `tof_YYYYMMDD_HHMMSS.bin`。
    - **路径选择**：通过文件对话框选择保存目录。

#### D. 数据回放 (Playback)
- **智能识别**：根据文件名识别数据类型和帧大小。
    - `depth_` 前缀 -> 解析为双通道 (64KB/帧)。
    - `tof_` 前缀 -> 解析为 ToF (32KB/帧)。
- **播放控制**：加载文件、播放、暂停、进度条显示。
- **流控制**：回放时自动暂停网络推流；停止回放后可恢复推流。

#### E. 离线重建 (Offline Reconstruction)
- **输入**：本地 ToF 原始数据文件 (.bin)。
- **算法支持**：
    - **峰值法 (Peak Detection)**：快速提取最强回波。
    - **匹配滤波 (Matched Filter)**：高信噪比重建 (Gaussian Smoothing 模拟)。
- **增强功能**：
    - **空间相关 (Spatial Correlation)**：支持 3x3 邻域直方图合并 (通过卷积模拟) 以提升信噪比。
- **输出**：重建后的强度像和距离像，直接显示在界面上。

## 3. 界面设计 (UI)

### 3.1 风格 (Theme)
- **Fusion Dark**：采用深灰色背景 (#2d2d2d) 和浅灰色文字 (#dcdcdc)。
- **配色**：蓝色 (#2a82da) 作为强调色，用于按钮选中、进度条和链接。
- **控件**：扁平化 GroupBox，现代 Tab 样式。

## 4. 技术栈
- **语言**：Python 3.x
- **GUI**：PyQt5
- **绘图**：PyQtGraph
- **科学计算**：NumPy, SciPy (用于卷积和滤波)
