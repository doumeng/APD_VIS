# 上位机开发计划 - UDP 热力图/图像显示

## 目标
开发一个基于 UDP 协议接收 128x128 像素的单光子激光雷达数据 (50Hz)，并以双视窗 (强度图 + 距离图) 实时显示的 Windows 上位机。

## 关键技术点
1.  **UDP Socket**：监听端口接收原始二进制流。
2.  **二进制解析**：解析自定义协议包头，分离强度通道和距离通道数据。
3.  **高性能绘图 (PyQtGraph)**：
    - 使用 `ImageItem` 快速渲染 128x128 灰度/伪彩色图。
    - 必须达到 50Hz 刷新率而不卡顿。
4.  **数据录制**：
    - 仅在用户点击"开始录制"时，将接收到的原始二进制数据追加写入 `.bin` 文件。
    - 包含简单的文件头或元数据 (可选)。
